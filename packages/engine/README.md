# @formsmithapp/engine

Framework-agnostic core engine for Formsmith — the block-tree state machine, navigation,
visibility / jump / calculation, validation orchestration, and answer piping. Runs identically in
the browser, Node, and at the edge, with **zero runtime dependencies**.

> 🚧 **Scaffolding only.** Implementation has not started yet; the public API will be documented
> here and at [formsmith.app](https://formsmith.app) as it lands.
